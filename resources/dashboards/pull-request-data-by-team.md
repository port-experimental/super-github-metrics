# Pull Request data by Team
Display pull request information, grouped by teams

## Prerequisite
- Teams should be ingested into Port
- Ownership between team, repositories, and pull requests should be set

## Steps
- Create a widget and select "Table"
- Blueprint: Pull Request
- After creation, click on "Customize table" at the 3 dots located at the top right corner
- Click on the "group" icon which are three squares layered on each other. Group by owning teams
